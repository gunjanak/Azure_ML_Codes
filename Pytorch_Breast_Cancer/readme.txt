The following project contains 5 files.
1. train.py (main code to get the data, train the model using pytorch, log parameters, and register the model)
2. environment.yml (defining the environment)
3. run.ipynb (connect to workspace, create a compute instance, use it to train model from the train.py)
4. score.py (support for deployment)
5. deployment.ipynb (deploy the registered model and test it)
